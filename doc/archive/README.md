# Archive Index

`doc/archive/*`는 현재 구현 기준이 아닌 과거 완료/이력 문서를 보관한다.

사용 규칙:

1. 신규 구현/운영 판단 기준으로 사용하지 않는다.
2. 과거 결정 배경, 마이그레이션 이력, 단일 일자 로그 확인에만 사용한다.
3. active 문서와 충돌하면 `AGENTS.md`와 `doc/planning/*`, `doc/reference/*` active 문서를 우선한다.

현재 보관 문서:

- `doc/archive/reference/core-addon-separation-remediation.completed.md`
- `doc/archive/operations/daily-log-2026-04-24.completed.md`
