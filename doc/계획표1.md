# 웹OS 프론트엔드 UI/UX 작업 계획표 (Glassmorphism 유지)

현재 적용된 **글래스모피즘(Glassmorphism) 스타일**을 기본 테마로 유지하면서, 기획서(v3.0)의 핵심 기능들을 완성하기 위한 작업 순서입니다.

## 1단계: 인증 및 보안 기반 구축 (Authentication)
*가장 먼저 시스템 접근을 제어하는 화면과 로직을 완성합니다.*

- [x] **로그인 화면 (Login UI) 고도화**
  - 글래스모피즘 기반의 반투명 로그인 모달 창 디자인.
  - 비밀번호 입력 및 JWT 토큰 처리 로직 연동 (현재 생성된 `Login.svelte` 보완).
  - 인증 상태에 따른 화면 전환 로직 (비로그인 시 데스크탑 접근 차단).
- [x] **Path Guard 시각적 피드백**
  - 권한이 없는 디렉토리(`/home/inri` 외부) 접근 시 나타날 `Toast` 알림 디자인.

## 2단계: 핵심 데스크탑 셸 완성 (Desktop Shell)
*앱들이 실행될 바탕화면과 창 관리 환경을 다듬습니다.*

- [x] **바탕화면 아이콘 리팩토링**
  - 앱 런처 아이콘(File Station, Terminal, Resource Monitor 등) 디자인 적용.
  - 마우스 호버(Hover) 시 부드러운 애니메이션 효과 추가.
- [x] **윈도우 매니저 (Window Manager) 디테일 향상**
  - 드래그 및 리사이즈 성능 최적화.
  - 창 활성화 시 z-index 최상위 호버링 효과(Box-shadow 및 강한 Blur 효과).
  - **애니메이션**: 창 마이그레이션(최소화/최대화) 시 부드러운 Scale 트랜지션 추가.
- [x] **태스크바 (Taskbar) 및 상태 표시줄**
  - 실행 중인 앱 아이콘 표시 및 클릭을 통한 창 최소화/복원.
  - 시스템 타이머 및 간단한 알림(Notification) 컴포넌트 우측 하단 배치.

## 3단계: 필수 내장 앱 구현 (Core Apps)
*백엔드 API와 연동하여 실제 데이터를 뿌려주고 제어하는 UI를 만듭니다.*

- [x] **File Station (파일 탐색기) 완성**
  - 그리드 뷰(Grid)와 리스트 뷰(List) 전환 UI.
  - 우클릭 커스텀 컨텍스트 메뉴 (새 폴더, 삭제, 이름 변경).
  - API 연동: 실제 `fs.js` 라우트를 통한 파일 목록 조회 및 CRUD 반영.
- [x] **Monaco 기반 Code Editor 통합**
  - 텍스트 파일 더블클릭 시 Monaco Editor가 탑재된 새 창 팝업.
  - 저장 단축키(Ctrl+S) 이벤트 및 저장 완료 토스트 알림 연동.
- [x] **실시간 Terminal 연동**
  - xterm.js UI 커스터마이징 (글래스모피즘 창 내부 렌더링).
  - WebSocket 스트리밍을 통한 실제 명령어 입출력 테스트.

## 4단계: 모니터링 데스크보드 (Resource Monitor)
*차트와 그래픽 요소가 많이 들어가는 시각적 대시보드를 구축합니다.*

- [x] **전체 UI 레이아웃**
  - 좌측 탭(Overview, CPU, Memory, Storage, Network, GPU) + 우측 상세 화면 분할 레이아웃.
- [x] **실시간 차트 적용**
  - Chart.js(Filler 포함)를 활용하여 1초 주기 REST 폴링으로 CPU 메트릭 시각화.
  - 어두운 글래스모피즘 테마에 맞는 네온 톤의 그래픽 적용.
- [x] **상세 수치 컴포넌트**
  - CPU/GPU 온도, 네트워크 트래픽 등 숫자 기반 메트릭을 보여주는 미니 카드 컴포넌트.

## 5단계: 시스템 유틸리티 및 마감 (Utilities & Polish)
*사용성을 극대화하는 디테일 요소들을 추가합니다.*

- [x] **Docker / 시스템 서비스 매니저 UI**
  - Docker 컨테이너 목록 조회, 시작/정지/재시작/삭제 기능 구현.
- [x] **글로벌 단축키 (Accessibility)**
  - ESC(창 닫기), Delete(창 닫기) 키보드 단축키 구현.
- [x] **전체 버그 수정 및 디자인 최적화**
  - 창이 화면 밖으로 나가지 않도록 한계선(Boundary) 설정.
  - GPU 가속 기반 translate3d 창 이동 최적화 적용.

## 6단계: 고급 멀티태스킹 및 클라우드 연동 (Advanced UI & Cloud)
*UI 편의성과 외부 저장소 확장성을 확보합니다.*

- [x] **인터랙티브 멀티태스킹**
  - [x] **Window Snapping**: 화면 가장자리 드래그 시 윈도우 분할 배치 기능.
  - [x] **Virtual Desktops**: 작업 공간 전환을 위한 가상 데스크톱 환경 구현.
- [x] **클라우드 스토리지 서비스 연동**
  - [x] **rclone 통합**: 구글 드라이브(Google Drive) 등 40+ 클라우드 API 연동 지원 확보.
  - [x] **Cloud Manager**: 설정 앱 내에서 클라우드 계정 관리 및 마운트 UI 구현.
- [x] **Spotlight 시스템 전역 검색**
  - [x] `Ctrl + Space` 단축키로 호출되는 앱 및 기능 통합 검색기 구현.
  - [x] 키보드 내비게이션 및 실행 흐름 최적화.

## 7단계: 미디어 지원 및 통합 뷰어 (Media & Viewers)
*외부 앱 없이 OS 내에서 대부분의 파일을 바로 확인하는 환경을 구축합니다.*

- [x] **통합 미디어 플레이어**
  - 오디오 미니 플레이어 및 비디오 웹 스트리밍 엔진 탑재.
  - 고해상도 이미지 전용 갤러리 앱.
- [x] **3D 모델 뷰어 (Three.js)**
  - `gltf`, `fbx`, `obj` 파일을 실시간으로 렌더링하고 조작하는 프리뷰 기능.
- [x] **오픈소스 오피스/에셋 뷰어**
  - `PDF.js` / Browser PDF (PDF)
  - `psd.js`를 이용한 디자인 에셋 미리보기 지원.

## 8단계: 시스템 제어 및 에코시스템 (System Core & Ecosystem)
*설정의 GUI화와 외부 확장성을 완성합니다.*

- [ ] **중앙 알림 센터 & 제어판**
  - 백그라운드 이벤트 및 경고를 관리하는 우측 알림 패널.
  - `.env` 수정 없이 GUI에서 테마, 보안 설정을 변경하는 통합 제어판.
- [ ] **보안 및 전송 강화**
  - **보안 폴더**: 개별 폴더 단위 암호화 잠금 기능.
  - **WebRTC 무선 전송**: 로컬 네트워크 기기 간 직접 파일 전송 기능.
- [ ] **위젯 및 바탕화면 커스터마이징**
  - 바탕화면에 고정 가능한 날씨, 리소스 등 미니 위젯 시스템.
  - 배경화면 변경 및 테마 컬러 커스텀 기능.
